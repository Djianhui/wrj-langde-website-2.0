# 无人机系统组件说明

## 功能特性

基于 TechnologyView 组件创建的新的无人机系统组件，包含以下特性：

### 1. 组件结构
- **文件位置**: `src/views/DroneSystemView.vue`
- **路由配置**: `/drone-system`
- **导航链接**: 已添加到主导航菜单

### 2. 系统模块
包含4个主要的无人机系统：

1. **监控无人机系统** (surveillance)
   - 4K超高清监控
   - 实时图像传输
   - 长续航设计
   - 抗干扰能力

2. **巡逻无人机系统** (patrol)
   - 自主导航
   - AI智能识别
   - 智能充电
   - 云端管理

3. **救援无人机系统** (rescue)
   - 热成像夜视
   - 物资投送
   - 抗恶劣天气
   - 通信中继

4. **巡检无人机系统** (inspection)
   - 精密检测
   - 电力专用模块
   - 数据记录分析
   - 智能维护

### 3. 页面内容
- **头部横幅**: 无人机系统介绍
- **系统展示**: 每个系统的详细介绍
- **技术优势**: 4大核心优势
- **性能指标**: 关键技术参数
- **合作伙伴**: 技术合作展示
- **联系咨询**: 解决方案咨询

### 4. 响应式设计
- 支持桌面端、平板端、手机端
- 动态语言切换（中文/英文）
- 交互动画效果

## 访问方式

您可以通过以下方式访问无人机系统页面：

1. **直接URL访问**: `http://localhost:3001/drone-system`
2. **导航菜单**: 点击顶部导航栏的"无人机系统"链接
3. **移动端菜单**: 通过移动端菜单访问

## 技术实现

- **Vue 3**: 使用Composition API
- **响应式设计**: 支持多端适配
- **国际化**: 支持中英文切换
- **动画效果**: CSS3动画和过渡
- **模块化设计**: 易于维护和扩展
